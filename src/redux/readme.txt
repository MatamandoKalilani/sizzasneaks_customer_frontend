,This folder "src\redux" contains all the logical code that this application uses to create and update the state of the SIZZASNEAKS e-commerce application.

The state of the application is kept in a redux store and each component in the application can access any state it needs from the store. Only one store is being used along with multiple reducers. 

The following is a breakdown of all the folder's contents:-

----------------------------------------------------------------------------------------------------------
src\redux\index.js

This file contains the code that creates a redux store to keep state for the SIZZASNEAKS application.

----------------------------------------------------------------------------------------------------------
src\redux\reducers

This folder contains the file that deals with the reducers for this application main redux Store.


----------------------------------------------------------------------------------------------------------
src\redux\actions

This folder contains files that make up redux actions that can be executed when dispatched.
